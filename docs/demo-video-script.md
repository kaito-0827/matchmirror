画面収録: macOS `cmd+shift+5`（画面全体 or Chromeウィンドウ）。マイクONで一発録りでOK。

## 0:00–0:20 課題提起（ランディング表示）
- https://matchmirror.vercel.app を表示
- 台詞: 「早期離職の最大の原因は、求人票と入社後の実態のギャップです。
  MatchMirrorは、このズレをAIエージェントが入社前に可視化するサービスです。」

## 0:20–1:00 候補者診断（チャット画面）
- 「無料で相性診断を始める」→ 企業を選択 → チャットで2〜3問答える
- 台詞: 「候補者はチャットで希望や不安を話すだけ。CandidateAgentが
  価値観・働き方のシグナルを抽出していきます。」

## 1:00–1:50 診断レポート
- レポート画面（6軸スコア・ズレ・根拠evidence）をスクロールしながら
- 台詞: 「MismatchAgentが企業の実態データと候補者の回答を6軸で照合。
  スコアだけでなく、企業側の記述と候補者の発言を並べた根拠付きで
  ズレを提示します。出力はGuardrailAgentが差別的・断定的表現を検査済みです。」

## 1:50–2:40 Autopilot（ここが審査基準①の見せ場）
- レポート下部の「AIに自律実行してもらう」ボタンをクリック
- 判断理由（🤖 reasoning）・優先質問・フォロー計画が一気に表示されるところを見せる
- 台詞: 「ここからが自律エージェントの真価です。OrchestratorAgentが診断結果を読み、
  『確認質問を先にすべきか、フォロー計画を直接作るべきか』を自分で判断。
  重点軸に沿って質問を並べ替え、FollowUpAgentが入社前後のフォロー計画まで
  一気通貫で生成します。人間はボタンを1回押すだけです。」

## 2:40–3:00 まとめ
- アーキテクチャ図（docs/assets/architecture.png）を表示
- 台詞: 「React + Cloud Run + Gemini API + Firestoreで構成。
  候補者と企業の双方が『入社前に確認すべきこと』を知れる世界を目指します。」
